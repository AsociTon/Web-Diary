Ambulance Alert System

What is Ambulance Alert System?

This project is about reducing ambulance commute time , by making an ambulance alert system. In this we implemented google api's to calculate position of ambulance when emergency system is on and accordingly we notify vehicles around it that are using any navigation system to alert them beforehand, so they can adjust and leave a save passage to the ambulance on the side guided by the the notification, thus we have a whole lane of vehicles (using navigation system) cordinating with each other to let an ambulance pass by easily, this created passage on one side of the road is made totally free of any confusion or noise.This will directly reduce the commute time of a patient. We also implemented a reassurance system i.e smart contract ,by which we recheck and verify if the ambulance which is not on duty doesn't misuse use this system and bothers the public unnecessarily.
